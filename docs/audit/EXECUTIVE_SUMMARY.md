## Appendix: Glossary

**E-E-A-T:** Experience, Expertise, Authoritativeness, Trustworthiness (Google's quality signals)

**Core Web Vitals:** Google's user experience metrics (LCP, FID/INP, CLS)

**LCP:** Largest Contentful Paint - Time for main content to load

**TBT:** Total Blocking Time - Time main thread is blocked

**CLS:** Cumulative Layout Shift - Visual stability score

**Featured Snippet:** Google's highlighted answer box at top of search results

**Schema Markup:** Structured data that helps search engines understand content

**AI Bot:** Automated crawler from AI companies (GPTBot, PerplexityBot, etc.)

**Topic Cluster:** Hub-and-spoke content organization strategy

**Hreflang:** HTML tag indicating alternate language versions

**SSR/SSG:** Server-Side Rendering / Static Site Generation (Next.js features)
