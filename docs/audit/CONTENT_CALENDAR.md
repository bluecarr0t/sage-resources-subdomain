## ðŸ“‹ Content Template Checklist

Every new page should include:

### Required Elements
- [ ] **Title** (50-60 characters)
- [ ] **Meta Description** (150-160 characters)
- [ ] **Keywords** (5-10 target keywords)
- [ ] **Hero Section** (headline + subheadline)
- [ ] **Quick Answer Box** (for AI bots)
- [ ] **TL;DR Section** (3-5 key points)
- [ ] **Main Content** (3-6 sections, 1,500-2,500 words)
- [ ] **FAQ Section** (5-10 questions minimum)
- [ ] **Key Takeaways** (5-7 points)
- [ ] **CTA Section** (contact/consultation)
- [ ] **Related Content Links** (3-5 internal links)

### Schema Markup
- [ ] **Article Schema** (always)
- [ ] **FAQPage Schema** (if FAQs present)
- [ ] **HowTo Schema** (if step-by-step guide)
- [ ] **BreadcrumbList Schema** (always)
- [ ] **Person Schema** (author byline)

### SEO Elements
- [ ] **H1 tag** (one per page, includes primary keyword)
- [ ] **H2 tags** (3-6, includes secondary keywords)
- [ ] **H3 tags** (as needed for subsections)
- [ ] **Alt text** for all images (descriptive, includes keywords)
- [ ] **Internal links** (5-10 contextual links)
- [ ] **External citations** (2-5 authoritative sources)

### AI Optimization
- [ ] **Quick Answer** at top (30-50 words)
- [ ] **TL;DR bullet points** (easy to extract)
- [ ] **Structured lists** (numbered, bulleted)
- [ ] **Clear, quotable answers** to questions
- [ ] **Source attribution** for claims
