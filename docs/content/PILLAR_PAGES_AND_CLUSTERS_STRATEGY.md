# Pillar Pages & Topic Clusters Strategy for Sage Outdoor Advisory

**Goal:** Implement a comprehensive pillar page and topic cluster SEO strategy to establish topical authority, improve rankings, and create a logical content hierarchy that guides users through the buyer journey.

---

## üéØ Overview: What Are Pillar Pages & Topic Clusters?

### Pillar Pages
- **Comprehensive, authoritative guides** (3000-5000+ words) covering broad topics
- Act as the **main hub** for a topic cluster
- Target high-volume, competitive keywords
- Provide comprehensive overview of the entire topic
- Link to multiple supporting cluster pages

### Cluster Pages
- **Specific, focused pages** (1000-2500 words) on narrow topics
- Support and link back to pillar pages
- Target long-tail keywords with specific intent
- Provide detailed information on subtopics

### The Strategy
```
Pillar Page (Main Hub)
  ‚îú‚îÄ‚îÄ Cluster Page 1 (Subtopic)
  ‚îú‚îÄ‚îÄ Cluster Page 2 (Subtopic)
  ‚îú‚îÄ‚îÄ Cluster Page 3 (Subtopic)
  ‚îî‚îÄ‚îÄ Cluster Page N (Subtopic)
```

**Why This Works:**
- Establishes topical authority for specific subjects
- Creates natural internal linking structure
- Signals to search engines your expertise
- Improves user experience and navigation
- Helps capture both broad and specific search queries

## üîó Internal Linking Strategy

### Link Hierarchy

**From Pillar Pages:**
- Link to all relevant cluster pages
- Link to related glossary terms
- Link to related tools/calculators
- Link to case studies
- Link to root domain service pages

**From Cluster Pages:**
- Link UP to parent pillar page (prominent)
- Link to related cluster pages
- Link to relevant glossary terms
- Link to root domain service pages

**From Glossary:**
- Link to relevant pillar pages
- Link to related landing pages
- Link to root domain content

### Link Placement

1. **In Content** (Contextual)
   - Natural, helpful links within content
   - Example: "Learn more about [glamping feasibility studies](/landing/glamping-feasibility-study)"

2. **Related Content Section** (Bottom of Page)
   - "Related Guides" section linking to pillar/cluster pages
   - "Related Terms" section linking to glossary

3. **Breadcrumbs** (Navigation)
   - Show content hierarchy
   - Example: Home > Guides > Glamping Industry > Glamping Feasibility Study

4. **Table of Contents** (Long-form Content)
   - Anchor links within pillar pages
   - Helps navigation and creates internal links

   - 150-160 characters
   - Include primary keyword
   - Include CTA

3. **H1 Tag**
   - Single H1 per page
   - Include primary keyword
   - Clear and descriptive

4. **Heading Structure**
   - Logical H2/H3 hierarchy
   - Include related keywords
   - Use descriptive headings

5. **Content Length**
   - Minimum 3000 words for pillar pages
   - Comprehensive coverage

6. **Internal Linking**
   - Link to 10-20 cluster pages
