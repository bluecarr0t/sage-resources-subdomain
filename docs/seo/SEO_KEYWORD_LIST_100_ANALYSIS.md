## üìã Using This List in SEMrush

### Recommended Analysis Workflow

1. **Import into SEMrush**
   - Upload CSV file
   - Select location (United States recommended)
   - Review search volume and keyword difficulty

2. **Filtering Strategy**
   - **Search Volume:** Target 100-10,000+ monthly searches
   - **Keyword Difficulty:** 
     - Easy (0-40): Quick wins for new content
     - Medium (41-60): Target with quality content
     - Hard (61+): Long-term goals, use in pillar pages

3. **Priority Matrix**
   - **High Volume + Low Difficulty:** Immediate opportunities
   - **High Volume + High Difficulty:** Pillar page targets
   - **Low Volume + Low Difficulty:** Long-tail opportunities
   - **Low Volume + High Difficulty:** Skip or reconsider

4. **Content Mapping**
   - Map keywords to content type (glossary/landing page/guide)
   - Identify keyword clusters for pillar pages
   - Plan internal linking structure

5. **Competitor Analysis**
   - Check who ranks for each keyword
   - Analyze top-ranking content
   - Identify content gaps
