## Template Quick Reference

### Location Page Structure:
```
1. Hero: "[Service] in [State]"
2. "[State] [Service] Expertise"
3. "Why [State] is Ideal"
4. Benefits (state-specific)
5. CTA
```

### Problem/Solution Page Structure:
```
1. Hero: Question or solution headline
2. Problem definition
3. Solution overview
4. Step-by-step (HowTo schema)
5. FAQ section
6. CTA
```

### Cost Page Structure:
```

2. Price range overview
3. Factors affecting cost
```
