## Quality Assurance Checklist

For each new page, verify:

### Content Quality
- [ ] Unique, valuable content (no duplication)
- [ ] 800+ words (1,200+ preferred)
- [ ] Proper heading hierarchy (H1, H2, H3)
- [ ] Primary keyword in title and H1
- [ ] Natural keyword usage throughout

### SEO Elements
- [ ] Meta title (50-60 characters)
- [ ] Meta description (150-160 characters)
- [ ] Primary keyword included in meta tags
- [ ] 3-5 internal links to related pages
- [ ] Links to relevant glossary terms (2-3)
- [ ] Links to related pillar pages (1-2)
- [ ] Links to root domain service pages

### Technical Elements
- [ ] Structured data implemented (Service, FAQPage, HowTo as applicable)
- [ ] BreadcrumbList schema (automatic)
- [ ] Mobile-responsive design
- [ ] Fast page load time
- [ ] No duplicate content issues
- [ ] Proper URL structure
- [ ] Canonical URL set

### User Experience
- [ ] Clear, compelling headline
- [ ] Scannable content format
- [ ] FAQ section (where applicable)
- [ ] CTA section with clear action
- [ ] Related pages section
- [ ] Readable font sizes and spacing
