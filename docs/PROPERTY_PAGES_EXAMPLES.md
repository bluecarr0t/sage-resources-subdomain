## How Property Pages Are Generated

### 1. Unique Property Names
The system fetches all unique property names from the `sage-glamping-data` table:
- Property names are trimmed and deduplicated
- Null or empty property names are excluded
- Approximately 1,000+ unique property names

### 2. Slug Generation
Each property name is converted to a URL-safe slug:
- Lowercase conversion
- Special characters removed (e.g., `&` becomes removed)
- Spaces replaced with hyphens
- Multiple hyphens collapsed to single hyphen
- Example: `"Elk & Embers Resort"` → `"elk-embers-resort"`

### 3. Route Generation
Each unique property name generates:
- A route at `/property/[slug]`
- A page with all locations/units for that property name
- Static generation at build time for optimal performance
- Automatic inclusion in sitemap.xml

### 4. Content Display
Each property page shows:
- **Header Section:** Property name, location, rating
- **Photos:** Google Places photos with carousel navigation (if available)
- **Locations/Units:** All properties with the same name (grouped by location)
- **Details:** Property type, operating season, capacity, rates
- **Amenities:** Visual grid of available amenities
- **Contact:** Website link (prioritizes `google_website_uri`), phone number, map link

## SEO Features

Each property page includes:
- **Meta Tags:** Title, description, keywords
- **OpenGraph Tags:** For social media sharing with property photos
- **Structured Data:**
  - BreadcrumbList schema (Home / Map / Property Name)
  - LocalBusiness schema (address, coordinates, rating, photos, opening hours)
- **Canonical URL:** Prevents duplicate content issues
- **Mobile Optimized:** Responsive design for all devices

## Accessing Property Pages

### From Map Page
- Property pages can be linked from map markers (future enhancement)
- Direct navigation via breadcrumbs: Home → Map → Property Name

### Direct URLs
- Format: `https://resources.sageoutdooradvisory.com/property/[slug]`
- Example: `https://resources.sageoutdooradvisory.com/property/collective-governors-island`
- All URLs are lowercase with hyphens

### Sitemap
- All property pages are automatically included in `/sitemap.xml`
- Priority: 0.8
- Change frequency: Monthly
- Positioned after map page, before landing pages

## Page Structure

### Breadcrumb Navigation
```
Home / Map / [Property Name]
```

### Main Sections
1. **Hero Section**
   - Large photo carousel
   - Property name (H1)
   - Location information
   - Rating and review count
   - Contact buttons

2. **Locations/Units Section** (if multiple)
   - Grouped by location (city, state)
   - Each unit shows: site name, unit type, rates, map link

3. **Property Details Section**
   - Property type
   - Operating season
   - Year opened
   - Unit capacity
   - Minimum nights

4. **Amenities Grid**
   - Visual checkmarks for available amenities
   - Pool, hot tub, wifi, pets, restaurant, etc.

5. **Contact & Links**
   - Website link (Visit Website button)
   - Phone number (clickable tel: link)
   - View on Map link
